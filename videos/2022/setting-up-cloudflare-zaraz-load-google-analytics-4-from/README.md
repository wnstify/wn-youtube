Cloudflare Zaraz feature is a great service, that will help you serve custom scripts without changing code in your projects.

You can load any type of analytics or tools directly from Cloudflare.
For supported tools, please check:
https://developers.cloudflare.com/zaraz/reference/supported-tools/

To read more about Zaraz, visit:
https://www.cloudflare.com/products/zaraz/

Also you have ability to add custom html snippets with script tag.
