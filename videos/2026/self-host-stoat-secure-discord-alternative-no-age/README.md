Discord leaked 70K government IDs. Now they want facial scans. Here's how to self-host Stoat securely instead.

Stoat is the closest open-source alternative to Discord you can run on your own server. It looks identical, works the same way, and keeps your community's data on your infrastructure. Most setup guides skip the security entirely. This one doesn't. We walk through a pre-tested, hardened Docker Compose setup with non-root containers, network segmentation, Docker secrets, health checks, and a properly configured reverse proxy with WebSocket support. Everything is on our GitHub, ready to deploy.
