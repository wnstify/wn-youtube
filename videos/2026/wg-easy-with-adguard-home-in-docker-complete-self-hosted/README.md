Build a secure VPN with network-wide ad blocking using WG Easy and AdGuard Home. A modern, actively maintained alternative to Mistborn with better features and ongoing development.

Mistborn development has slowed significantly. You don't need to wait. WG Easy plus AdGuard Home gives you the same core functionality with cleaner interfaces and active communities. This tutorial walks you through the complete setup using Docker Compose.

Complete setup files and Docker Compose configuration available on GitHub (link below).
